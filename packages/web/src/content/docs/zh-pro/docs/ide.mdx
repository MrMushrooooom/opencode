opencode 可与 VS Code、Cursor 或任何支持终端的 IDE 集成。只需在终端中运行 `opencode` 即可开始使用。

---

## 使用方法

- **快速启动**：使用 `Cmd+Esc`（Mac）或 `Ctrl+Esc`（Windows/Linux）在分屏终端视图中打开 opencode，或激活已存在的运行会话。
- **新建会话**：使用 `Cmd+Shift+Esc`（Mac）或 `Ctrl+Shift+Esc`（Windows/Linux）启动新的 opencode 终端会话，（即使已有会话运行）。你也可以点击界面中的 opencode 按钮。
- **上下文感知**：自动与 opencode 共享你当前选中的内容或标签页。
- **文件引用快捷键**：使用 `Cmd+Option+K`（Mac）或 `Alt+Ctrl+K`（Linux/Windows）插入文件引用。例如，`@File#L37-42`。

---

## 安装

若要在 VS Code 和主流衍生版本（如 Cursor、Windsurf、VSCodium ）上安装 opencode，操作如下：

1. 打开 VS Code
2. 启动集成终端
3. 运行 `opencode` - 扩展将自动安装

---

### 手动安装

在扩展市场中搜索 **opencode** 并点击**安装**。

---

### 故障排查

若扩展无法自动安装：

- 确保你在集成终端中运行 `opencode`。
- 确认已安装对应 IDE 的 CLI 工具：
  -  VS Code：`code` 命令
  -  Cursor：`cursor` 命令
  -  Windsurf：`windsurf` 命令
  -  VSCodium：`codium` 命令
  - 若未安装，按 `Cmd+Shift+P`（Mac）或按 `Ctrl+Shift+P`（Windows/Linux）并搜索 "Shell Command: Install 'code' command in PATH"（或 IDE 的等效命令）
- 确保 VS Code 有安装扩展的权限
