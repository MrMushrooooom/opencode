opencode 使用 [AI SDK](https://ai-sdk.dev/) 和 [Models.dev](https://models.dev) 来支持 **75+ 个 LLM 提供商**，并且支持运行本地模型。

---

## 提供商

大多数热门提供商默认已预加载。如果你通过 `opencode auth login` 添加了某个提供商的凭据，当你启动 opencode 时该提供商就能使用。

了解更多关于[提供商]的信息(/zh/docs/providers)。

---

## 选择模型

配置好提供商后，你可以通过输入以下命令选择你想要的模型：

```bash frame="none"
/models
```

## 设置默认模型

要将其中一个设置为默认模型，你可以在 opencode 配置中设置 `model` 键。

```json title="opencode.json" {3}
{
  "$schema": "https://opencode.ai/config.json",
  "model": "lmstudio/google/gemma-3n-e4b"
}
```

这里的完整 ID 是 `provider_id/model_id`。

如果你配置了[自定义提供商](/zh/docs/providers#custom)，`provider_id` 是配置 `provider` 部分对应的键，`model_id` 是 `provider.models` 里对应的键。

---

## 加载模型

当 opencode 启动时，它会按以下优先级顺序检查模型：

1. --model 或 -m 命令行标记。格式与配置文件中的一致：provider_id/model_id 。

2. opencode 配置中的模型列表。

   ```json title="opencode.json"
   {
     "$schema": "https://opencode.ai/config.json",
     "model": "anthropic/claude-sonnet-4-20250514"
   }
   ```

   这里的格式是 `provider/model`。

3. 最后使用的模型。

4. 按内部优先级排序的首个模型。
