opencode [JS/TS SDK](https://www.npmjs.com/package/@opencode-ai/sdk) 提供了一个类型安全的客户端，用于与 opencode 服务器交互。您可以使用它来构建自定义集成并以编程方式控制 opencode。

[了解更多](/docs/server) opencode 服务器的工作原理。

---

## 安装

从 npm 安装 SDK：

```bash
npm install @opencode-ai/sdk
```

---

## 创建客户端

创建一个客户端实例来连接到您的 opencode 服务器：

```javascript
import { createOpencodeClient } from "@opencode-ai/sdk"

const client = createOpencodeClient({
  baseUrl: "http://localhost:4096",
})
```

#### 选项

| 选项            | 类型       | 描述                         | 默认值                  |
| --------------- | ---------- | ---------------------------- | ----------------------- |
| `baseUrl`       | `string`   | opencode 服务器的 URL        | `http://localhost:4096` |
| `fetch`         | `function` | 自定义 fetch 实现            | `globalThis.fetch`      |
| `parseAs`       | `string`   | 响应解析方法                 | `auto`                  |
| `responseStyle` | `string`   | 返回样式：`data` 或 `fields` | `fields`                |
| `throwOnError`  | `boolean`  | 抛出错误而不是返回           | `false`                 |

---

## 启动服务器

您也可以以编程方式启动 opencode 服务器：

````javascript
import { createOpencodeServer } from "@opencode-ai/sdk"

const server = await createOpencodeServer({
  hostname: "127.0.0.1",
  port: 4096,
})

console.log(`Server running at ${server.url}`)

server.close()
}

#### 选项

| 选项       | 类型          | 描述                    | 默认值      |
| ---------- | ------------- | ----------------------- | ----------- |
| `hostname` | `string`      | 服务器主机名            | `127.0.0.1` |
| `port`     | `number`      | 服务器端口              | `4096`      |
| `signal`   | `AbortSignal` | 取消操作的中止信号      | `undefined` |
| `timeout`  | `number`      | 服务器启动超时时间（毫秒） | `5000`      |

## 类型

SDK 包含所有 API 类型的 TypeScript 定义。直接导入它们：

```typescript
import type { Session, Message, Part } from "@opencode-ai/sdk"
````

所有类型都是从服务器的 OpenAPI 规范生成的，可在 <a href={typesUrl}>类型文件</a> 中找到。

---

## 错误

SDK 抛出类型化错误，您可以捕获和处理：

```typescript
try {
  const session = await client.session.get({ id: "invalid-id" })
} catch (error) {
  console.error("Failed to get session:", error.message)
}
```

## APIs

SDK 通过类型安全的客户端接口暴露所有服务器 API。

---

### App

| 方法         | 描述           | 响应                                    |
| ------------ | -------------- | --------------------------------------- |
| `app.get()`  | 获取应用信息   | <a href={typesUrl}><code>App</code></a> |
| `app.init()` | 初始化应用程序 | `boolean`                               |

---

#### 示例

```javascript
const app = await client.app.get()
await client.app.init()
```

---

### Config

| 方法                 | 描述                 | 响应                                                                                                  |
| -------------------- | -------------------- | ----------------------------------------------------------------------------------------------------- |
| `config.get()`       | 获取配置信息         | <a href={typesUrl}><code>Config</code></a>                                                            |
| `config.providers()` | 列出提供商和默认模型 | `{ providers: `<a href={typesUrl}><code>Provider[]</code></a>`, default: { [key: string]: string } }` |

---

#### 示例

```javascript
const config = await client.config.get()
const { providers, default: defaults } = await client.config.providers()
```

### Sessions

| 方法                                                          | 描述                       | 注释                                                                                                                 |
| ------------------------------------------------------------- | -------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `session.list()`                                              | 列出会话                   | 返回 <a href={typesUrl}><code>Session[]</code></a>                                                                   |
| `session.get({ id })`                                         | 获取会话                   | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.children({ id })`                                    | 列出子会话                 | 返回 <a href={typesUrl}><code>Session[]</code></a>                                                                   |
| `session.create({ parentID?, title? })`                       | 创建会话                   | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.delete({ id })`                                      | 删除会话                   | 返回 `boolean`                                                                                                       |
| `session.update({ id, title? })`                              | 更新会话属性               | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.init({ id, messageID, providerID, modelID })`        | 分析应用并创建 `AGENTS.md` | 返回 `boolean`                                                                                                       |
| `session.abort({ id })`                                       | 中止正在运行的会话         | 返回 `boolean`                                                                                                       |
| `session.share({ id })`                                       | 分享会话                   | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.unshare({ id })`                                     | 取消分享会话               | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.summarize({ id, providerID, modelID })`              | 总结会话                   | 返回 `boolean`                                                                                                       |
| `session.messages({ id })`                                    | 列出会话中的消息           | 返回 `{ info: `<a href={typesUrl}><code>Message</code></a>`, parts: `<a href={typesUrl}><code>Part[]</code></a>`}[]` |
| `session.message({ id, messageID })`                          | 获取消息详情               | 返回 `{ info: `<a href={typesUrl}><code>Message</code></a>`, parts: `<a href={typesUrl}><code>Part[]</code></a>`}`   |
| `session.chat({ id, ...chatInput })`                          | 发送聊天消息               | 返回 <a href={typesUrl}><code>Message</code></a>                                                                     |
| `session.shell({ id, agent, command })`                       | 运行 shell 命令            | 返回 <a href={typesUrl}><code>Message</code></a>                                                                     |
| `session.revert({ id, messageID, partID? })`                  | 回退消息                   | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.unrevert({ id })`                                    | 恢复已回退的消息           | 返回 <a href={typesUrl}><code>Session</code></a>                                                                     |
| `session.permissions.respond({ id, permissionID, response })` | 响应权限请求               | 返回 `boolean`                                                                                                       |

---

#### 示例

```javascript
// 创建和管理会话
const session = await client.session.create({ title: "My session" })
const sessions = await client.session.list()

// 发送消息
const message = await client.session.chat({
  id: session.id,
  providerID: "anthropic",
  modelID: "claude-3-5-sonnet-20241022",
  parts: [{ type: "text", text: "Hello!" }],
})
```

### Files

| 方法                      | 描述                 | 响应                                                                                |
| ------------------------- | -------------------- | ----------------------------------------------------------------------------------- |
| `find.text({ pattern })`  | 在文件中搜索文本     | 包含 `path`、`lines`、`line_number`、`absolute_offset`、`submatches` 的匹配对象数组 |
| `find.files({ query })`   | 按名称查找文件       | `string[]`（文件路径）                                                              |
| `find.symbols({ query })` | 查找工作区符号       | <a href={typesUrl}><code>Symbol[]</code></a>                                        |
| `file.read({ path })`     | 读取文件             | `{ type: "raw" \| "patch", content: string }`                                       |
| `file.status()`           | 获取已跟踪文件的状态 | <a href={typesUrl}><code>File[]</code></a>                                          |

---

#### 示例

```javascript
// 搜索和读取文件
const textResults = await client.find.text({ pattern: "function.*opencode" })
const files = await client.find.files({ query: "*.ts" })
const content = await client.file.read({ path: "src/index.ts" })
```
