---
title: MCP 服务器
description: 添加本地和远程 MCP 工具。
---

您可以使用 _模型上下文协议_（Model Context Protocol，MCP）向 opencode 添加外部工具。opencode 支持：

- 本地服务器
- 和远程服务器

添加后，MCP 工具会自动与内置工具一起提供给 LLM。

---

## 配置

您可以在 opencode 配置中的 `mcp` 下定义 MCP 服务器。

---

### 本地

在 MCP 对象中使用 `"type": "local"` 添加本地 MCP 服务器。可以添加多个 MCP 服务器。每个服务器的键字符串可以是任意名称。

```json title="opencode.json"
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "my-local-mcp-server": {
      "type": "local",
      "command": ["bun", "x", "my-mcp-command"],
      "enabled": true,
      "environment": {
        "MY_ENV_VAR": "my_env_var_value"
      }
    },
    "my-different-local-mcp-server": {
      "type": "local",
      "command": ["bun", "x", "my-other-mcp-command"],
      "enabled": true
    }
  }
}
```

您也可以通过将 `enabled` 设置为 `false` 来禁用服务器。如果您想临时禁用服务器而不从配置中删除它，这会很有用。

---

### 远程

在 `mcp` 下使用 `"type": "remote"` 添加远程 MCP 服务器。

```json title="opencode.json"
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "my-remote-mcp": {
      "type": "remote",
      "url": "https://my-mcp-server.com",
      "enabled": true,
      "headers": {
        "Authorization": "Bearer MY_API_KEY"
      }
    }
  }
}
```

本地和远程服务器可以在同一个 `mcp` 配置对象中一起使用。

```json title="opencode.json"
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "my-local-mcp-server": {
      "type": "local",
      "command": ["bun", "x", "my-mcp-command"],
      "enabled": true,
      "environment": {
        "MY_ENV_VAR": "my_env_var_value"
      }
    },
    "my-remote-mcp": {
      "type": "remote",
      "url": "https://my-mcp-server.com",
      "enabled": true,
      "headers": {
        "Authorization": "Bearer MY_API_KEY"
      }
    }
  }
}
```

## 按代理

如果您有大量 MCP 服务器，您可能希望只在每个代理中启用它们并在全局禁用它们。要做到这一点：

1. 配置 MCP 服务器。
2. 在全局将其作为工具禁用。
3. 在您的[代理配置](/docs/agents#tools)中将 MCP 服务器作为工具启用。

```json title="opencode.json" {11, 14-17}
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "my-mcp": {
      "type": "local",
      "command": ["bun", "x", "my-mcp-command"],
      "enabled": true
    }
  },
  "tools": {
    "my-mcp*": false
  },
  "agent": {
    "my-agent": {
      "tools": {
        "my-mcp*": true
      }
    }
  }
}
```
